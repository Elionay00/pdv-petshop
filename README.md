# 🐾 PDV PET SHOP - Sistema de Auto Atendimento

## Descrição
Sistema PDV (Ponto de Venda) desenvolvido para Pet Shop com interface gráfica moderna usando CustomTkinter.

## Funcionalidades
- Cadastro de produtos com categorias
- Consulta de produtos (busca por ID/nome e filtro por categoria)
- Visualização de detalhes do produto
- Tema claro/escuro (salvo automaticamente)

## Tecnologias
- Python 3.x
- CustomTkinter (interface gráfica)
- SQLite3 (banco de dados)
- JSON (configurações)

## Como executar

1. Instalar dependências:
```bash
pip install -r requirements.txt
py main.py
